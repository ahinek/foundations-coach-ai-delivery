Foundations Coach for AI Delivery

This repository outlines a minimal, principle-driven view of a role I’ve been calling Foundations Coaching for AI Delivery.

It is not a framework.
It is not a certification.
It is not a replacement for Scrum Masters or Agile Coaches.

It is a starting point.

AI is now embedded in how teams deliver software.
When something that powerful becomes ubiquitous, the foundations beneath it matter more than ever.

This repository describes the guardrails, signals, and responsibilities that help ensure AI magnifies the best qualities of how teams work together — rather than amplifying fragmentation.

Why This Exists

AI accelerates delivery.

Acceleration is powerful — but only when built on:

Shared guardrails

Shared tooling ecosystems

Shared product clarity

Shared accountability

Preserved team cohesion

This repository outlines how a Foundations Coach might steward those elements in practice.

Who This Is For

Scrum Masters

Agile Coaches

Engineering Managers

Product Leaders

Delivery Leaders

Anyone stewarding AI adoption inside teams
